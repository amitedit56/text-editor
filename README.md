# 📝 Amit Text Editor

A simple and professional text editor built using Python and Tkinter.
This project demonstrates GUI development, file handling, and basic text editing features.

---

## 🚀 Features

* 📂 New, Open, Save, Save As files
* ✂️ Cut, Copy, Paste functionality
* ↩️ Undo / Redo support
* 🔍 Zoom In / Zoom Out
* 🖥️ Clean and minimal UI

---

## 🛠️ Tech Stack

* Python 3
* Tkinter (built-in GUI library)

---

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/amit-text-editor.git
```

2. Navigate to project folder:

```bash
cd amit-text-editor
```

3. Run the application:

```bash
python main.py
```
